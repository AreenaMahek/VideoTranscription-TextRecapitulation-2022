About this Application

This Text Summarization tool uses T5, GPT, Autotokenizer with DistalBart,BART models for generating abstract versions of the original text.This tool also performs web scraping by compaction and paraphrasing the information from the websites and creates a short and a coherant version of this information. It compares all these summaries to distinguish which of the summary is an accurate one.
