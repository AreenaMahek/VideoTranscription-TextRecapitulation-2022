About this Application

This Text Summarization tool uses SpaCy,Gensim,NLTK,TF-IDF for generating abstract versions of the original text.This tool also performs web scraping by extracting the information from the websites and creates a short and a coherant version of this information. It compares all these libraries to allow the users to pick a summary of their choice as per their requirement.
